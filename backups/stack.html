1. Fundamentos Básicos de IA: Lo que la Gente Entiende Poco

La Inteligencia Artificial (IA) no es un "robot mágico"... Piensa en IA como un "aprendiz superdotado": le das instrucciones claras (prompts), y usa patrones aprendidos para producir outputs, pero siempre necesita validación humana para evitar errores.

1.1 Qué son los Tokens (Explicación Simple, Profunda y Práctica)

Tokens son las "unidades básicas de pensamiento" de la IA: cada palabra o parte de palabra que procesa... (ej. "inteligencia" = "intel" + "igencia").

Cómo Funcionan:

Input prompt → IA divide en tokens → Procesa modelo (predice siguiente token) → Genera output...

Cálculo Práctico y Ejemplos:

Usa Tokenizer OpenAI (web) o Tiktoken (Python). Ejemplo:

  1. Prompt: "Genera WBS app PMO 20 tareas." ≈ 500 tokens input.
  2. Respuesta: Lista detallada ≈ 800 tokens output.
  3. Total: ≈ 1300 tokens.
  4. Costo GPT-4o: Input $0.005/1K = $0.0025; Output $0.015/1K = $0.012; Total ≈ $0.0145.

Tip: Prompts cortos (<500 tokens) baratos; largos (contexto) precisos pero caros.

1.2 Costos Detallados (Fijos vs. Por Tokens)

  • Fijos: Suscripción mensual (ChatGPT Plus $20/mes). Ventaja: Predictible. Desventaja: Pagas aunque uses poco.
  • Por Tokens: Pago por uso (OpenAI GPT-4o: $0.005/1K input, $0.015/1K output; Claude: ~$0.01/1K combinado). Ventaja: Solo pagas lo usado. Desventaja: Menos predictible sin medir.

Calculadora: Prompt 1000 palabras (~1300 tokens) + respuesta 1000 = ~2600 tokens ≈ $0.026.

Tip: Usa dashboards API (OpenAI/Claude) para track; resume contexto para ahorrar.

1.3 Tipos de IA (Conversacionales vs. Agenticas y Más, con Analogías)

  • Conversacionales (LLMs): "Chat con experto" (ChatGPT). Ayuda: Brainstorm, docs.
  • Agenticas: "Agentes secretos con misión" (CrewAI). Ayuda: Automatiza flujos complejos.
  • CLI: "Ninjas en terminal" (Gemini CLI). Ayuda: Ops PMO batch.
  • Multimedia: "Artistas digitales" (Runway). Ayuda: Prototipos, demos.
  • Asistentes Código/Ext VS Code: "Gadgets en tu taller" (Copilot). Ayuda: Desarrollo fluido.
  • IAs Chinas: "Traductores culturales globales" (Ernie). Ayuda: Diversidad, global.
  • Otras Versátiles: Kilo Code (refactor local), Manus AI (edición docs), Perplexity (búsqueda real-time), etc.

1.4 Cómo Empezar con IA (Checklist Interactiva Detallada para Principiantes)

  • [ ] **Aprende Tokens:** Prueba tokenizer OpenAI (web gratis).
  • [ ] **Setup Básico (1h):** Regístrate ChatGPT free, instala VS Code + Codeium + Continue.dev (gratis). Descarga Ollama (local gratis).
  • [ ] **Prueba Conversacional:** ChatGPT "Genera idea PMO app salud, explica tokens usados".
  • [ ] **Prueba Agentica:** Instala LangChain (pip gratis), prueba loop simple.
  • [ ] **Mide Costos:** Calcula costo prompt típico (~$0.01-0.05).
  • [ ] **Evita Errores:** Corta prompts (<500 tokens gratis), limita loops agents (max 5), especifica rol, valida outputs.
  • [ ] **Integra Múltiples:** Prueba chain simple (ChatGPT idea → Claude diseño → Cursor código). Mide tokens total.
  • [ ] **Para PMO Práctica:** Asigna IAs por fase (ver secciones stacks).
